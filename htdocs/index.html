Delivery of pbf tiles as static hosting

You have "htdocs" folder for static hosting. If your vector tiles(pbf) are stored in "htdocs" folder, the VT can be accessible at:
http://localhost:3000/VTpractice/1/1/1.pbf

For example, you can get pbf tile at:
./VTpractice/1/1/1.pbf



Check the Map with the VT

you can check the map with this HTML file:
./map/map-pbf.html
